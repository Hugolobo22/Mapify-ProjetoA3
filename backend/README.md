# Site Turismo - Mapify


## Requisitos
- Node.js (v18+ recomendado)
- NPM ou Yarn


## Instalação
1. Clone ou crie a pasta do projeto e cole os arquivos conforme a estrutura acima.
2. No terminal, rode:


```bash
npm install
```


3. Crie um arquivo `.env` e cole os dados fornecidos pelo dono do projeto (Consulte-o para pegar as chaves):


4. Inicie a aplicação em desenvolvimento:


```bash
npm run dev
```
